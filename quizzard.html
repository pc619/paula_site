Quizzard

Computing 2 | University Project

Summary

Brief

To make a web app of our choice using functional and asynchronous programming.

To achieve the brief above, I decided to make a tool for my mom, a primary school teacher. The web app is a simple quiz maker for teachers that makes lessons (specially remote) interactive. Once teachers have created a quiz, they can click on it to generate a code for a life session.

Students can input this code into the home screen, then input their names and be added to the game. When all students have joined, the teacher can begin the live quiz! When everyone has finished, students are ranked based on how quick they were and how many answers they got right... Quizzard brings the competition to remote lessons in a user-friendly, simple way.
